==========================================
🧠 Data Mining Dashboard - README
==========================================

📦 Welcome! This is a standalone, local machine learning web app that allows you to:

✅ Train a model using your own CSV data  
✅ Predict outcomes using a trained model  
✅ Visualize decision trees and results  
✅ View interactive analytics and export predictions  

---------------------------------------------------
🖥️ How to Run (Non-Developers)
---------------------------------------------------

1. Double-click `dist/start_app.exe`

⚠️ NOTE: This file may be flagged by your antivirus (e.g., McAfee or Windows Defender) because it is a custom-built executable and unsigned.

🔐 This app is 100% safe to use.

➡️ If blocked:
- Add the app folder to your antivirus exclusions.
- You can usually do this from:
  "Virus & Threat Protection" → "Protection History" → "Allow on Device"

🟢 Once allowed, your browser will open automatically to:
http://localhost:5000

---------------------------------------------------
💻 How to Run (Python Developers)
---------------------------------------------------

1. Make sure Python 3.9+ is installed.
2. Open a terminal or command prompt inside the project folder.
3. Install required libraries:

    pip install -r requirements.txt

4. Run the app locally:

    run_app.bat

🧠 This runs `start_app.py` and automatically opens the app in your browser.

---------------------------------------------------
📂 Folder Structure (main ones)
---------------------------------------------------

- `dist/start_app.exe`      → Pre-packaged executable version  
- `start_app.py`            → Python entry point  
- `app.py`                  → Main application logic  
- `templates/`              → Web UI (HTML files)  
- `static/`                 → CSS, JavaScript, images  
- `sample/To_Train/`        → Example training CSV files  
- `sample/To_Predict/`      → Example prediction CSV files  
- `requirements.txt`        → List of Python dependencies  
- `run_app.bat`             → Easy launcher for developers
